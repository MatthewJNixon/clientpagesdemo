Welcome to Matt's Developer Onboarding

Overview

https://matts-fhir.com is a fully R4-compliant FHIR Server capable of processing the following FHIR Resources:

  • Patient
  • Practitioner
  • CareTeam

Onboarding

  • Get Credentials: Register with the FHIR server admin to obtain an API key or OAuth credentials.
  • Attach to APIM Subscription Key: Ensure you attach your API key for secure access.
